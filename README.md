# Java用户管理系统（GUI）

一个基于Java Swing教学开发模仿的用户管理系统GUI练习项目，支持用户增删、登录验证的基础功能。

## 📋 功能介绍

- 🔑 简单的用户登录验证

- 📝 用户信息增删

- 🎨 基础GUI界面美化（自定义布局/按钮）

- 📦 可直接运行的绿色版程序（无需安装JRE）

## 🚀 快速开始

### 方式1：直接运行（Windows）

1. 下载Release中的`ManageDemoApp.zip`；

2. 解压后双击`ManageDemoApp.exe`即可运行（内置JRE，免安装）。

(PS : 使用登录名`admin`和密码`123456`即可登录)

### 方式2：源码运行（开发/调试）

1. 确保本地安装OpenJDK 22+、IntelliJ IDEA；

2. 克隆仓库：`git clone https://github.com/Tsinkwatree/JavaGUI-imitation-learning-employee-sys.git`；

3. 导入IDEA，运行主类`APP.java`。

## 🛠️ 技术栈

- Java SE 22（OpenJDK）

- Swing（GUI开发）

- jpackage
